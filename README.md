**IKS Invisble Economy**

An interactive research dashboard and AI-powered intelligence suite focused on the Indigenous Knowledge Systems (IKS) of South Africa. This application visualizes the "Invisible Economy"—the estimated 9.8 million practitioners excluded from the formal economy—and provides tools to bridge the gap between indigenous wisdom and modern policy.

🚀 **Key Features**

Interactive Workforce Visualization: Dynamic Chart.js dashboard showing the scale of IKS sectors (Zulu 2023 data).

Cost Dimension Analysis: Tabbed exploration of Economic, Cognitive, and Opportunity costs of non-recognition.

✨ **IKS Intelligence Suite (Gemini AI Powered)**

Sector Impact Simulator: Socioeconomic projections for recognition.

Policy Bridge Drafter: Sui Generis legislative framework generation.

Knowledge Synergy Engine: Bridging IKS with global challenges.

Biopiracy Risk Scanner: Identifying misappropriation of indigenous resources.

Education Designer: Creating "Interfaced" lesson plans for cognitive justice.

🛠️ **Technical Stack**

Frontend: HTML5, Tailwind CSS

Visualizations: Chart.js

AI Engine: Google Gemini 2.5 Flash API

Deployment: Optimized for GitHub Pages
